Stack used: JDK 8, SpringBoot, JUnit, Mockito

Make sure that JAVA_HOME is alredy set!!

Run the shell script 'boot.sh' to start the SpringBoot application.

Visit the Swagger URL 'http://localhost:8080/coffeehouse/swagger-ui.html' to check all the available REST APIs.

Some sample data for Coffee variety and Order has already been inserted.
